⚠️ Running as a local file — Plex sync won't work
Run python -m http.server 8080 in the app folder, then open http://localhost:8080 — or double-click START-REELBOX.bat.
👋 First-time setup — Enter your GitHub and TMDB tokens below to enable cross-device sync and movie discovery. These are stored only in this browser, never in the code.

🔑 GitHub Sync Token

Required for syncing your ratings and watch statuses across devices. Stored only in your browser — never in the code.

Go to GitHub → Settings → Developer Settings → Personal Access Tokens → Tokens (classic). Create a token with repo scope checked.

🖥️ Plex Connection

Connect to your Plex Media Server to import your full movie library with cover art, ratings, cast, and descriptions.

Your server's local IP — find it in Plex Settings → Remote Access.
Plex Web → play any movie → ••• menu → Get Info → View XML → copy token from URL. Full guide →
Routes requests through your Worker to bypass CORS. Leave blank if running locally. Get a free Worker →

🎬 TMDB API Key

For Discover — upcoming movies, now playing, and search. Free at themoviedb.org. Stored only in your browser.

📱 Install as App

Add ReelBox to your home screen for a native app experience — works on iPhone, Android, and desktop.

On iPhone/iPad: tap the Share button (□↑) in Safari, then tap "Add to Home Screen".
✅ ReelBox is installed as an app on this device!

🗂️ Data

Watch statuses and library data are stored locally in your browser and synced to GitHub.
